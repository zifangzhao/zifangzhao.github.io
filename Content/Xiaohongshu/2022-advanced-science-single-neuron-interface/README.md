# 2022 Advanced Science - Surface Interface for Human Single Neurons

Source PDF: `../../../Resources/Hassan_2022_AdvSci_Translational Organic Neural Interface Devices at Single Neuron Resolution.pdf`

## Core Message

This paper shows that conformable organic neural interface devices can record human cortical activity with enough spatial and temporal detail to identify individual units from the brain surface, without penetrating the tissue. The study reports 229 putative single units and uses their properties to study human cortical microcircuit activity.
