# WILD Datalogger Manuscript - Wireless Modular Neuro-Behavioral Recording

Source PDF: `../../../Resources/Zhao_datalogger_MS_revision5_main_only.pdf`

## Core Message

This manuscript presents WILD, a Wireless, Interactive, Lightweight Datalogger for small animals. It integrates flexible neural probes, optogenetics, an inertial measurement unit, ultrasonic microphone, and head-mounted camera. The platform supports long-term multimodal recordings from freely moving mice, including group and outdoor settings, and uses onboard model-based real-time processing to trigger closed-loop interventions.
