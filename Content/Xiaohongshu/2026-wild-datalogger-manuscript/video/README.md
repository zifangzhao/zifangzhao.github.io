# 竖屏短视频说明

- 时长：约 40 秒
- 画幅：1080 x 1440 JPG 轮播图合成
- 当前版本：无配音，可直接用作节奏预览；配音可使用父目录 README 的脚本录制。
- 字幕：`subtitles.srt`
- 视频：`short_video.mp4`

## 每页节奏

1. 自然行为中的大脑活动，怎么记录？ - 理解自然行为，需要更自由的记录技术。
2. 传统系统很难同时满足这些要求 - 小动物设备必须轻、无线、多模态、可闭环。
3. WILD: Wireless, Interactive, Lightweight Datalogger - 多模态同步是研究复杂行为的基础。
4. 大脑、运动、叫声、眼动一起看 - 行为不是单一变量，神经数据也不该孤立解读。
5. 设备还能实时判断并触发干预 - WILD 的目标是记录自然行为，也能在自然行为中做因果干预。
