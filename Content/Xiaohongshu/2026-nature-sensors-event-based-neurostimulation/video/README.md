# 竖屏短视频说明

- 时长：约 40 秒
- 画幅：1080 x 1440 JPG 轮播图合成
- 当前版本：无配音，可直接用作节奏预览；配音可使用父目录 README 的脚本录制。
- 字幕：`subtitles.srt`
- 视频：`short_video.mp4`

## 每页节奏

1. 设备能不能只在事件发生时耗能？ - 低功耗闭环需要事件驱动。
2. 传统系统强大，但不够柔软省电 - 柔性、生物兼容和高速低功耗要同时实现。
3. OECN 事件传感器 - 有机传感器也可以高速响应。
4. 每个 spike 约 40 pJ - 闭环设备的核心指标之一是每次响应消耗多少能量。
5. 检测异常放电，并触发刺激 - 事件传感器可以成为下一代低功耗闭环神经调控的一部分。
