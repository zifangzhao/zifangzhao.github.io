# 2026 Nature Sensors - Organic Event-Based Sensors for Closed-Loop Neurostimulation

Source PDF: `../../../Resources/Yang_2026_NS_High frequency, low energy organic event based sensors for closed loop neurostimulation.pdf`

## Core Message

This paper develops organic electrochemical neuron (OECN)-based event sensors for closed-loop neurostimulation. The sensors respond within about 1 ms, generate pulses up to 1.1 kHz, cover mammalian neural activity bandwidth from 0.5 to 1,000 Hz, and consume about 40 pJ per spike. They detect hippocampal interictal epileptiform discharges and drive real-time stimulation that suppresses pathological sleep spindle oscillations in vivo.
