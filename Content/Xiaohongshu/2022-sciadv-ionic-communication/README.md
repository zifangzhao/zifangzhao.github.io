# 2022 Science Advances - Ionic Communication

Source PDF: `../../../Resources/Zhao_2022_SciAdv_Ionic communication for implantable bioelectronics.pdf`

## Core Message

Implanted devices need to send data through tissue. This paper introduces ionic communication (IC), a communication strategy that uses ionic/polarizable biological media to propagate megahertz-range signals, enabling stable data transmission from implanted bioelectronics without relying only on conventional RF-style approaches.

## Audience Hook

身体组织不是通信障碍，它也可以成为通信介质。

Title: 传统方案在体内都有代价

Copy: 线缆会穿过组织；传统无线方式又常受功耗、刚性器件和耦合条件限制。

Takeaway: 体内通信必须顺着身体的物理环境来设计。

### Page 3 - The Idea

Title: 把组织本身变成通信通道

Visual: Ionic communication mechanism figure.

Copy: 离子通信利用极化介质传播 MHz 信号，让组织不再只是障碍，而是可利用的通道。

Takeaway: 这是一种从组织物理出发的通信方式。

### Page 4 - In Vivo Result

Title: 自由活动动物里的长期神经数据传输

Visual: Implanted device and neural recording result.

Copy: 基于 IC 的设备在自由活动动物中连续数周稳定传输神经数据，质量足以分离单神经元动作电位。

Takeaway: 离子通信把“传不出来”变成了“可以长期稳定传出”。
