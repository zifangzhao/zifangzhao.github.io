# 竖屏短视频说明

- 时长：约 36 秒
- 画幅：1080 x 1440 JPG 轮播图合成
- 当前版本：无配音，可直接用作节奏预览；配音可使用父目录 README 的脚本录制。
- 字幕：`subtitles.srt`
- 视频：`short_video.mp4`

## 每页节奏

1. 植入式设备为什么难把数据传出来？ - 通信是植入式设备真正的瓶颈之一。
2. 传统方案在体内都有代价 - 体内通信必须顺着身体的物理环境来设计。
3. 把组织本身变成通信通道 - 这是一种从组织物理出发的通信方式。
4. 自由活动动物里的长期神经数据传输 - 离子通信把“传不出来”变成了“可以长期稳定传出”。
