# 2024 Advanced Science - Anisotropic Conducting Interlayer for Epidermal EMG

Source PDF: `../../../Resources/Zhao_2024_AS_Formation of Anisotropic Conducting Interlayer for High Resolution Epidermal.pdf`

## Core Message

This paper develops biocompatible anisotropic conducting interlayer composites (ACI) for high-resolution epidermal electrophysiology. The main problem is that conventional conductive pastes can create lateral ionic crosstalk between electrodes, reducing spatial resolution. ACI improves dense cutaneous EMG recordings and enables single muscle action potentials with distinct spatial profiles.

### Page 3 - The Solution

Title: ACI: 只让信号按需要的方向传

Copy: ACI 的目标很明确：让信号主要垂直通过，而不是在表面横向扩散。

Takeaway: 界面材料可以设计信号传播方向。
