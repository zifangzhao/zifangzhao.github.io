# 竖屏短视频说明

- 时长：约 40 秒
- 画幅：1080 x 1440 JPG 轮播图合成
- 当前版本：无配音，可直接用作节奏预览；配音可使用父目录 README 的脚本录制。
- 字幕：`subtitles.srt`
- 视频：`short_video.mp4`

## 每页节奏

1. 皮肤表面记录，也可以看到更细的肌肉信号 - 非侵入记录的关键不只是电极，也包括电极和皮肤之间的界面。
2. 导电膏会让相邻通道"串味" - 低阻抗不等于高分辨率。
3. ACI: 只让信号按需要的方向传 - 界面材料可以设计信号传播方向。
4. 单个肌肉动作电位的空间图谱 - 看清空间结构，才能更好理解运动系统。
5. 从小鼠、灵长类到人类 - 高分辨率表皮电生理可服务疾病诊断和人机交互。
