# 2023 Nature Materials - vIGT Stand-Alone Organic Bioelectronics

Source PDF: `../../../Resources/Cea_2023_NatMat_Integrated internal ion gated organic electrochemical transistors for standalone conformab[42843].pdf`

## Core Message

This paper presents vertical internal ion-gated organic electrochemical transistors (vIGTs) that support megahertz-range operation in densely packed arrays without crosstalk. The work uses these devices to build conformable, fully organic circuits that can acquire, process, power, and wirelessly communicate neurophysiological signals in freely moving rodents.
