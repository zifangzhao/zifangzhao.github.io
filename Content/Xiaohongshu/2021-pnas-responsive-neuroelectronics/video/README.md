# 竖屏短视频说明

- 时长：约 40 秒
- 画幅：1080 x 1440 JPG 轮播图合成
- 当前版本：无配音，可直接用作节奏预览；配音可使用父目录 README 的脚本录制。
- 字幕：`subtitles.srt`
- 视频：`short_video.mp4`

## 每页节奏

1. 大脑异常信号出现时，设备能不能自己反应？ - 闭环神经接口 = 记录 + 判断 + 刺激。
2. 通道越多，电子系统越难小型化 - 高通道数和低功耗之间需要新的架构。
3. MTA: 先复用，再放大 - 系统架构本身就是神经接口创新。
4. 不只是记录，还能实时干预 - 设备从"传感器"变成了"响应式系统"。
5. 用闭环刺激改变病理性脑网络耦合 - 神经接口的目标不是替代大脑，而是理解并调节异常网络。
